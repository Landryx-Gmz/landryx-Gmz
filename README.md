## Sobre mí

Soy desarrollador de software con base sólida en **Python** y **TypeScript**. Diseño sistemas con **Clean Architecture**, aplico seguridad desde el inicio y uso la IA no solo como producto final, sino como herramienta en cada fase del desarrollo — desde el diseño hasta el despliegue.

Mi enfoque: construir software que funcione, escale y se mantenga.
